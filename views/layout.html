Sip.js

This sip button will execute a javascript function using the SIP.js API. This call right now is hard-coded to go to Asterisk @50.56.219.107 which then redirects the call to my cell phone.

For some reason, if the call is not answered, the server crashes. I believe this is because it does not receive the proper disconnect agreement from asterisk. It also has something to do with port numbers. Like a port is already in use.

USB Camera

This button runs a child process command which finds a specific USB id. If the id exists, it then runs the service motion start command which starts streaming the USB camera on a local IP address

JSSIP

JSSIP is another Javascript SIP library which tries to implement websockets

SIPml5

SIPml5 is a client side way of making phone calls. In order to actually make a call to asterisk, you need to use the webrtc2sip server side thing and have it configured with asterisk. Webrtc2sip is the middle man which handles the codec stuff.
